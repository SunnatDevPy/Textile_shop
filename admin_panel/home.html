Главная страница для показа клиенту

Быстрый сценарий показа

1) Откройте клиентский dashboard для KPI и складских метрик.
2) Покажите страницу платежей и поток Click/Payme.
3) Перейдите в админку для CRUD и управления заказами.

Куда перейти

Выберите нужную часть системы для демонстрации.

Что видно клиенту

- Продажи и KPI
- Склад: остатки и общая стоимость товара
- Разрез по типу одежды: мужская / женская

Что видно администратору

- Управление товарами, заказами, каталогами
- Поиск и фильтры
- История, статистика, платежные callback test
